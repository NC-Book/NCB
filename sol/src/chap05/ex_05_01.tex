
\exercise{Averages revision \label{exSWaverage}}{1}
Compute the averages of the following sequences of numbers: a) $3,7,5$;  b) $1,3,3,1,3,3,1,1,1$.

\solution
We find the solution by adding up all the numbers and dividing by the number of numbers that we added up. This means for part (a)
\eq{
\frac{3+7+5}{3} = \frac{15}{3}=5
}
For part (b) we can save us some work by using multiplication to add up the numbers rather than adding them up term by term 
\eq{
\frac{4\cdot 3+5\cdot 1}{9}= \frac{17}{9}=1.\overline{8}
}
