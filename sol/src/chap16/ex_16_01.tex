\exercise{Geometric dynamics\label{exBrainGeo}}{1}
Starting from the initial condition $\delta(0)=\delta_0$, integrate 
\eqn{
\dot{\delta} = a\delta^2.
}

\solution 
We separate the variables, by writing the differential equation as 
\eq{
   \delta^{-2} \d \delta = a \d t. 
}
Then we integrate 
\eqa{
 \int  \delta^{-2} \d \delta &=& \int  a \d t. \\{} 
 -  \delta^{-1}  &=& at +C  \\
   \delta^{-1}  &=& -at-C \\
}
which we can solve for
\eq{
\delta(t)=\frac{1}{-at-C}.
}
To fix the constant of integration we recall the initial condition  
\eq{
   \delta_0 = \delta_{0}  = \frac{1}{-C}
}
which implies 
\eq{
-C = {\delta_0}^{-1}
}
Hence we can write the solution to the initial value problem as 
\eq{
\delta(t) = \frac{1}{{\delta_0}^{-1}-at}
}
