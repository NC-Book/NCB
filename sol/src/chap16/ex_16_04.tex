\exercise{Degeneracy of the epidemic threshold\label{exBrainDegen}}{2}
Consider the SIS model in the form 
\eqn{
\dot{x}=f(x,p) = px(1-x) - x
}
where we have used $x$ instead of $I$ as the variable. Find the epidemic threshold, and then check the genericity conditions of the 
fold bifurcation $f_{\rm p}\neq 0$, $f_{\rm xx}\neq 0$. 

\solution 
We consider the steady state $x^*=0$. To find the epidemic threshold, our bifurcation point, we consider the Jacobian 
\eq{
f_{\rm x} = p - 2px -1 
}
To restrict the Jacobian to the steady state we substitute $x^*=0$ and then look for the point where the eigenvalue crosses 0, which yields
\eq{
p-1 = 0
}
So the invasion threshold is at $p^*=1$. Now we can evaluate the genericity conditions
\eq{
f_{\rm xx}(x^*,p^*) = \left. -2p \right|_{x=0,p=1} = -2 \neq 0,
}
\eq{
f_{\rm p}(x^*,p^*) = \left. x-x^2 \right|_{x=0,p=1} = 0.
}
This shows that the second condition, the so-called \emph{transversality condition} of the bifurcation theorem is violated.
