\exercise{Fold bifurcation normal form\label{exBrainFold}}{2}
Compute the steady states and their stability for the system
\eqn{
\dot{x} = a x^2 + b p. 
}


\solution 

We compute the steady states by solving 
\eq{
0=a\delta^2+b\rho
}
which yields 
\eq{
x^*=\pm \sqrt{-\frac{bp}{a}}
}
This means we have either two steady states ($b\rho/a<0$) or none ($b\rho/a>0$). If $a$ and $b$ have the same sign then steady states exist at negative values of $\rho$, otherwise they only exist at positive values of $\rho$. 

In this sense there are two forms of the fold bifurcation, but they are indistinguishable in practice.

To determine the stability of solutions we compute
\eq{
f_{\rm x}=\frac{\partial }{\partial x} ax^2+bp = 2ax
}
If $a>0$ then the positive steady state is unstable and the negative one is unstable. For $a<0$ it's the other way around.

\end{center}
This diagram corresponds to the case $a<0$ (positive branch of steady states is stable) and $b>0$ (steady states exist for negative $p$).
