\exercise{SIS on a three node chain \label{exHeatSISchain}}{1} 
In the chapter we introduced the interacting particle notation for the SIS model master equation. Use your intuition to formulate the Jacobian of the master equation for the SIS model on a chain of three nodes, (1)-(2)-(3). 

\solution 
We take the following into account:
\begin{itemize}
\item Node 1 can recover: ${\bf R} \otimes {\bf I} \otimes {\bf I}$
\item Node 2 can recover: ${\bf I} \otimes {\bf R} \otimes {\bf I}$
\item Node 3 can recover: ${\bf I} \otimes {\bf I} \otimes {\bf R}$
\item Node 1 can infect node 2: ${\bf Q} \otimes {\bf P} \otimes {\bf I}$
\item Node 2 can infect node 1: ${\bf P} \otimes {\bf Q} \otimes {\bf I}$
\item Node 2 can infect node 3: ${\bf I} \otimes {\bf Q} \otimes {\bf P}$
\item Node 3 can infect node 2: ${\bf I} \otimes {\bf P} \otimes {\bf Q}$
\end{itemize}
Hence the Jacobian is 
\eqa{
{\bf J} &=& {\bf R} \otimes {\bf I} \otimes {\bf I} + {\bf I} \otimes {\bf R} \otimes {\bf I} + {\bf I} \otimes {\bf I} \otimes {\bf R} \\
& & + {\bf Q} \otimes {\bf P} \otimes {\bf I} + {\bf P} \otimes {\bf Q} \otimes {\bf I} \\
& & + {\bf I} \otimes {\bf Q} \otimes {\bf P} + {\bf I} \otimes {\bf P} \otimes {\bf Q}
}
