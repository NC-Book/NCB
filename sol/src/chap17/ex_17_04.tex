\exercise{Linear manifold example\label{exManifoldInvariant}}{2} 
Use eigendecomposition to find the invariant manifolds of the steady state in the following linear system:
\eqn{
\begin{array}{r c l}
\dot{x} &=&2x+4y\\
\dot{y} &=&x-y, 
\end{array}
}


\solution 
Because the system is linear we know that the trajectories have the form 
\eq{
\vec{x}(t) = c_1 \vec{v_1} \exp{\lambda_1 t} + c_2 \vec{v_2} \exp{\lambda_2 t} 
}
hence we need to compute the eigenvectors and eigenvalues of the Jacobian matrix 
\eq{
{\bf J} = \avecc{2 & 4 \\ 1 & -1}. 
}
We find the characteristic polynomial as 
\eq{
0 = \left| \avecc{2-\lambda & 4 \\ 1 & -1- \lambda} \right|  
}
which yields 
\eqa{
0&=&(2-\lambda)(-1-\lambda)-4 \\
0&=&\lambda^2 - \lambda - 6 \\ 
0&=&(\lambda - 1/2)^2 - 1/4 - 6 \\
0&=&(\lambda - 1/2)^2 - 25/4 \\
0&=& \lambda - 1/2 \pm 5/2  \\
}
Hence we find the eigenvalues 
\eq{
\lambda_1 = 3 \qqq \lambda_2 = -2
}
We now determine the eigenvectors in the usual way. First for the eigenvalue 
$\lambda_1=3$:
\eq{
\avecc{2 & 4 \\ 1 & -1}\avec{1\\ a} = \avec{3 \\ 3a}
}
The lower row reads 
\eqa{ 
1-1a &=& 3a \\
a &=& 1/4
}
multiplying the whole vector by 4 we write 
\eq{
\vec{v_1} = \avec{4\\ 1} 
}
Using the same approach for the second eigenvector yields
\eq{
\avecc{2 & 4 \\ 1 & -1}\avec{1\\ a} = \avec{-2 \\ -2a}
}
and hence 
\eqa{
1-a &=& -2a \\ 
 a &=& -1
}
which gives us the eigenvector 
\eq{
\vec{v_2} = \avec{1 \\ -1}
}
We now know that trajectories have the form 
\eq{
\vec{x}(t) = c_1 \avec{4 \\ 1} \exp{3t} + c_2 \avec{1 \\ -1} \exp{-2t} 
}
If our initial value is such that $c_1=0$ we are on a trajectory
\eq{
\vec{x}(t) =  c_2 \avec{1 \\ -1} \exp{-2t} 
}
that approaches the state (0,0) and hence we are on the stable manifold. We can also write this as 
\eqa{
x &=& c_2 \exp{-2t} \\
y &=& -c_2 \exp{-2t}
}
for all values of $c_2$ and $t$. Substituting the first equation into the second we find 
\eq{
y=-x
}
which describes the manifold that we are looking for. In the same the unstable manifold consists of the points visited by trajectories for which $c_2=0$, so 
\eq{
\vec{x}(t) =  c_1 \avec{4 \\ 1} \exp{3t}
}
which means  
\eqa{
x &=& 4c_1 \exp{3t} \\
y &=& c_1 \exp{3t}
}
such that 
\eq{
y = \frac{1}{4}x 
}
is the unstable manifold we are looking for.
