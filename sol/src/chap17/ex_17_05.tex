\exercise{Manifold approximation}{2}
Find a second order (quadratic) approximation to invariant manifolds of the steady state at $(0,0)$ in the following system:
\eqan{
\dot{x} &=& 2x + 4y -x^2 \\
\dot{y} &=& x - y +xy
}


\solution 
We choose the Ansatz
\eq{
x=ay+by^2+O(y^3)
}
Differentiating the Ansatz yields 
\eq{
\dot{x}=(a+2by)\dot{y} +O(2)
}
Now we substitute the equations of motion from the exercise 
\eq{
2x + 4y -x^2 = (a+2by)(x - y +xy) + O(3)
}
We can now substitute the Ansatz itself. 
\eq{
2x + 4(ax+bx^2) -x^2 = (a+2b(ax+bx^2))(x - (ax+bx^2) +x(ax+bx)) + O(3)
}
We need to multiply all these terms out. However, everything that contains $x^3$ can be immediately be absorbed int the $O(3)$ terms. 
\eqa{
2x + 4ax+4bx^2 -x^2 &=& (a+2abx)(x - ax-bx^2 +ax^2) + O(3) \\
2x + 4ax+4bx^2 -x^2 &=& ax -a^2x-abx^2+a^2x^2 +2abx^2 -2a^2bx^2 + O(3) \\
(2+3a+a^2)x  &=& (a^2+2ab-2a^2b-ab-4b+1) x^2 + O(3)
}
Since the equation must be true for every $x$ (all the way along the manifold) the coefficients of the $x$ and $x^2$ terms must be zero. We first consider the coefficient of $x$ 
\eqa{
a^2+3a+2&=&0 \\
(a+3/2)^2-9/4+2&=& 0\\
(a+3/2)^2&=& 1/4\\
a+3/2&=& \pm 1/2\\
a = -3/2 \pm 1/2
}
So $a$ is $-1$ or $-2$ which one should we chose? Actually recall from the previous exercise (or compute it now) that the stable eigenvector was $(-1,1)^{\rm T}$. The manifold must be tangential to it and hence we need a function that has slope $-1$ in $x=0$ to point in the direction of the eigenvector. So it has to have the form $x=-y+O(y^2)$. This means we could have inferred $a=-1$ just from the previously computed eigenvector, but its a good check to to see it appear also in this way. So now we know for sure $a=-1$. Let's substitute it in the coefficient of $x^2$, 
\eqa{ 
0&=&a^2+2ab-2a^2b-ab-4b+1 \\
0&=&1-2b-2b+b-4b+1 \\
0&=&2-7b \\
b&=&2/7 
}
In summary our approximation to the manifold is
\eq{
x=-y+\frac{2}{7}y^2
}
If you are wondering if this is really the stable manifold and not the unstable one, you can substitute this into the equation for $\dot{y}$:
\eq{
\dot{y} = x - y +xy = -2y + O(y^2)
}
Hence close to the steady state at zero the equation behaves like 
\eq{
\dot{y}=-2y 
}
which shows that it is stable. 

The unstable manifold can be computed in the same way. The result is 
\eq{
x=4y - 4y^2+O(y^3) 
}
