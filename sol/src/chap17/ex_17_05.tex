\exercise{Manifold approximation}{2}
Find a second order (quadratic) approximation to the invariant manifolds of the steady state at $(0,0)$ in the following system:
\eqan{
\dot{x} &=& 2x + 4y -x^2 \\
\dot{y} &=& x - y +xy
}


\solution 
We choose the Ansatz
\eq{
x=ay+by^2+O(y^3)
}
Differentiating the Ansatz yields 
\eq{
\dot{x}=(a+2by)\dot{y} +O(y^3)
}
Now we substitute the equations of motion from the exercise 
\eq{
2x + 4y -x^2 = (a+2by)(x - y +xy) + O(y^3)
}
We can now substitute the Ansatz itself. 
\eq{
2(ay+by^2) + 4y -(ay+by^2)^2 = (a+2by)((ay+by^2) - y +(ay+by^2)y) + O(y^3)
}
We need to multiply all these terms out. However, everything that contains $y^3$ can be immediately be absorbed into the $O(y^3)$ terms. 
\eqa{
2ay+ 2by^2+4y-a^2y^2 &=& (a+2by)(ay+by^2-y+ay^2) + O(y^3) \\
2ay+ 2by^2+4y-a^2y^2 &=& a^2y+aby^2-ay+a^2y^2+2aby^2-2by^2 + O(y^3) \\
(3a+4-a^2)y  &=& (2a^2+3ab-4b) y^2 + O(y^3)
}
Since the equation must be true for every $y$ (all the way along the manifold) the coefficients of the $y$ and $y^2$ terms must be zero. We first consider the coefficient of $y$ 
\eqa{
a^2-3a-4&=&0 \\
(a-4)(a+1)&=& 0
}
So $a$ is $-1$ or $4$ which one should we chose? Actually recall from the previous exercise (or compute it now) that the stable eigenvector was $(-1,1)^{\rm T}$. The manifold must be tangential to it and hence we need a function that has slope $-1$ in $x=0$ to point in the direction of the eigenvector. So it has to have the form $x=-y+O(y^2)$. This means we could have inferred $a=-1$ just from the previously computed eigenvector, but it's a good check to to see it appear also in this way. So now we know for sure $a=-1$. Let's substitute it in the coefficient of $y^2$, 
\eqa{ 
0&=&2a^2+3ab-4b\\
0&=&2-3b-4b \\
0&=&2-7b\\
b&=& \frac{2}{7}
}
In summary our approximation to the manifold is
\eq{
x=-y+\frac{2}{7}y^2
}
If you are wondering if this is really the stable manifold and not the unstable one, you can substitute this into the equation for $\dot{y}$:
\eq{
\dot{y} = x - y +xy = -2y + O(y^2)
}
Hence close to the steady state at zero the equation behaves like 
\eq{
\dot{y}=-2y 
}
which shows that it is stable. 

The unstable manifold can be computed in the same way. The result is 
\eq{
x=4y - 4y^2+O(y^3) 
}
