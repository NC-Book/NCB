\exercise{Simple conservation law}{2}
If you did the exercises of the previous chapters you might be able to guess the conservation law found in the dynamical system 
\eqn{
\dot{x} = y \qqq \dot{y}=-x
}
Can you also find the conservation law mathematically?

\solution 
This system featured in some previous exercises where we showed that the trajectories are set of concentric circles around zero. So we might guess that the conservation law has the form
\eq{
x^2 + y^2 = r^2
}
where $r$ is the constant radius of the circle. If we already know this it is tempting to convert to polar coordinates, which yields the equation $\dot{r}=0$ which proves that the radius is conserved. However in this case a simpler approach that does not require prior knowledge of the solution is to use the method that was presented in this chapter. We write 
\eq{
\frac{\d x}{\d y } = \frac{\dot{x}}{\dot{y}} = -\frac{y}{x}
}
The equation 
\eq{
\frac{\d x}{\d y } = -\frac{y}{x}
}
can be separated into 
\eq{
x \d x = -y \d y 
}
which we can integrate 
\eqa{
\int x \d x &=& - \int y \d y  \\
\frac{1}{2} x^2 &=& - \frac{1}{2} y^2 + C 
}
which we can also write as 
\eq{
x^2+y^2 = 2C
}
which is the same result as we guessed above, except that we named the conserved property $2C$ instead of $r^2$, but that doesn't change anything.
