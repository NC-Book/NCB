\exercise{Leaky conveyor}{4}
Consider a random walker on a network described by the a weighted, directed adjacency matrix 
\eq{
{\bf A} = \aveccc{ 0 & 1 & 0 \\ 2 & 0 & 1 \\ 1 & 2 & 0}
}
the element $A_{ij}$ describes the rate at which the walker that is in node $j$ jump to node $i$. Determine the probability that the walker is in node 1 in the stationary state. [The probabilities become stationary, not the walker.] 

\solution 
The differential equations that describe this system are 
\eqa{
\dot{x}_1 &=& -3x_1 + x_2 \\
\dot{x}_2 &=& -3x_2 +2x_1 + x_3 \\
\dot{x}_3 &=& -x_3 + x_1 + 2x_2 
}
which we can write in matrix form 
\eq{
\dot{\vec{x}} = -{\bf L} \vec{x} 
}
where 
\eq{
\bf{L} = \aveccc{ 3 & -1 & 0 \\ -2 & 3 & -1 \\ -1 & -2 & 1}
}
To find the long term behavior we need to find the eigenvector with eigenvalues 0. (There needs to be one otherwise the walker number would not be conserved). 
Since we already know that the eigenvelue we are looking for is zero we can move straight to the conditions 
\eqa{
0 &=& -3v_1 + v_2 \\
0 &=& -3v_2 +2v_1 + v_3 \\
0 &=& -v_3 + v_1 + 2v_2 
}
Of course this is the same as finding the steady state of the dynamical system, just with $v$ instead of $x$. The first equation tells us 
\eq{
v_2 = 3 v_1
}
which we can use to eliminate $v_2$,
\eqa{
0 &=& -9v_1 +2v_1 + v_3 = -7v_1 +v_3 \\
0 &=& -v_3 + v_1 + 6v_1 = 7v-1 - v_3
}
Hence the eigenvector is 
\eq{
\vec{v_1} = \avec{1 \\ 3 \\ 7} 
}
Since we are talking about probabilities we want to normalize the vector such that is sums to one. So, after a long time the probability of finding the walker in node is just 1/11.
