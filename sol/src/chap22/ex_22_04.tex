

\exercise{Abstract stability analysis}{2}
We place copies of a dynamical system into the nodes of a network and couple them diffusively such that the the local within-node Jacobian and the coupling matrix are given by
\eq{
{\bf P}=\avecc{2 & -3 \\ 3 & -4}, \hspace{1cm}
{\bf C} = \avecc{0 & 0 \\ 0 & 1}
}
Compute the master stability function for this system. Describe what the result means for the network. 

\solution
Using the master stability function trick we know that we can compute the eigenvalues of the Jacobian as eigenvalues of  
\eq{
{\bf P} - \kappa  {\bf C} =  \avecc{2 & -3 \\ 3 & -4-\kappa} 
}
The characteristic polynomial of this matrix is 
\eqa{
0 &=& \lambda^2+(\kappa+2)\lambda+(1-2\kappa)  \\
0 &=& (\lambda+(\kappa+2)/2)^2 + (1-2\kappa)-(\kappa+2)^2/4 \\
(\lambda+(\kappa+2)/2)^2  &=&  -(1-2\kappa)+(\kappa+2)^2/4 \\
(2\lambda+(\kappa+2))^2  &=&  -4(1-2\kappa)+(\kappa+2)^2 \\
(2\lambda+(\kappa+2))^2  &=&  \kappa^2+12\kappa \\
2\lambda+(\kappa+2)  &=&  \pm \sqrt{\kappa^2+12\kappa} \\
2\lambda  &=&-(\kappa+2)  \pm \sqrt{\kappa^2+12\kappa} 
}
and hence
\eq{
\lambda_{1,2} = \frac{-(\kappa+2)  \pm \sqrt{\kappa^2+12\kappa}}{2} 
}
and 
\eq{
M(\kappa) =  \frac{-(\kappa+2) + \sqrt{\kappa^2+12\kappa}}{2} 
}
The homogeneous state is unstable if there is a $\kappa$ such that $M>0$. To find the bifurcation points where this happens we can solve 
\eqa{
0 &=& -(\kappa+2) + \sqrt{\kappa^2+12\kappa} \\ 
\kappa+2 &=&  \sqrt{\kappa^2+12\kappa} \\
\kappa^2+4\kappa+4 &=& \kappa^2+12\kappa \\
4 &=& 8 \kappa 
}
Hence the threshold is at 
\eq{
\kappa= \frac{1}{2}
}
which implies that the homogeneous steady state is unstable if there is a Laplacian eigenvalue $\kappa\geq 1/2$. In an unweighted network this is always the case if the network contains a link. However, the homogeneous steady state could still be stable on a weighted network if the links are very light.
