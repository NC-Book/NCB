\subquestion Suppose the adjacency matrix has an eigenvalue $\vec{v}$ with eigenvalue $\lambda$. Normalize the first element of this eigenvector to $v_1=1$, then determine the eigenvector elements $v_2$, $v_3$ and $v_4$. For which values of $\lambda$ is $v_3>0$? 

\solution 
We start from the eigenvector equation 
\eq{
{\bf A} \vec{v} = \lambda \vec{v}.
}
Node 1 has only one neighbor, node 2, hence the first row of $\bf A$ only contains one non-zero element, $A_{1,2}=1$. Hence, the first row of the eigenvector equation reads
\eq{
A_{1,2} v_2 = \lambda v_1. 
}
Using $A_{1,2}=1$ we can write this as 
\eq{
v_2 = \lambda v_1 = \lambda.
}
The second row of the eigenvector equation has the form 
\eq{
A_{2,1}v_1 + A_{2,4} v_4 = \lambda v_2, 
}
and hence 
\eq{
v_1 + v_4 = \lambda v_2
}
Substituting $v_1=1$ and $v_2=\lambda$ yields 
\eq{
1 + v_4 = \lambda^2   
}
and hence 
\eq{v_4=\lambda^2-1}

To find $v_3$ we consider the third row of the Eigenvector equation
\eq{
A_{3,4}v_4 = \lambda v_3.
}
Solving for $v_3$ we find
\eq{
v_3 = \frac{v_4}{\lambda} = \frac{\lambda^2 - 1}{\lambda} 
}
The element $v_3$ is positive when 
\eq{
\frac{\lambda^2 - 1}{\lambda} > 0
}
If $\lambda$ is positive this yields the condition
\eq{
\lambda>1
}
For the case $\lambda<0$ we multiply both sides with $\lambda$, 
\eqa{
\lambda^2 -1 &<& \lambda \\
\lambda^2 -\lambda -1 &<& 0 \\
(\lambda -1/2)^2 &<& 5/4 
}
Since we are looking for negative values of $\lambda$ we only need to consider the negative square root, which leads us to the golden ratio's little brother
\eq{
\lambda  > \frac{1-\sqrt{5}}{2}. 
}
In summary, if $\lambda>1$ then $v_3>0$ and in this case all of the elements of our four nodes are positive. The nodes in this sparsely connected branch find it difficult to sustain large eigenvalues, if other parts of the network want a large eigenvalue our four nodes have to pull all in the same direction to keep up.  

If $0<\lambda<1$ then $v_3<0$. Checking the other nodes reveals that the sign change occurs between node 3 and node 4. 

If $(1-\sqrt{5})/2< \lambda < 0$ then $v_3>0$, but $v_2<0$ and $v_4<0$.  

If $\lambda<(1-\sqrt{5})/2$ then then $v_3<0$. Now a sign chain occurs in every link of the branch. Since the branch is bipartite, alternating the signs in every step is a way to sustain relatively large negative eigenvalues. 

\subquestion
Consider also the case where instead of the branch a chain of 10 nodes is attached to the rest of the network. Compute the eigenvector elements for the nodes in the chain for the case $\lambda=9$. 

\solution 
Let's number the nodes consecutively from the tip (node 1) to end that is attached to the rest of the network (node 10). We start by choosing the normalization so that 
\eq{
v_1=1.
} 

Node one only has one neighbor, node 2, and hence the first row of the eigenvector equation for this system reads 
\eq{
v_2 = \lambda v_1 = 9 
}
The eigenvector equation for the second row reads 
\eqa{
v_1 + V_3 &=& \lambda v_2 \\
1 + v_3 &=& 81
}
so 
\eq{
v_3=80
}
From row three we get 
\eq{
v_2 + v_4 = \lambda v_2 
}
which we can write as 
\eq{
v_4 = 9 v_3 - v_2 
}
and generally 
\eq{
v_{i+1} = 9 v_i - v_{i-1} 
}
so 
\eqa{
v_4 &=& 9 v_3 - v_2 = 720 - 9 = 711 \\
v_5 &=& 9 v_4 - v_3 = 6399 - 80 = 6319  \\
v_6 &=& 9 v_5 - V_4 = 56871 - 711 = 56160 \\
v_7 &=& 9 v_6 - v_5 = 505440 - 6319 = 499121 \\
v_8 &=& 9 v_7 - v_6 = 4492089 - 56160 = 4435929\\
v_9 &=& 9 v_8 - v_7 = 39923361 - 499121 = 39424240\\
v_{10} &=& 9 v_9 - v_8 = 354818160 - 4435929 = 350382231 \\
v_{11} &=& 9 v_{10} - v_9 = 3153440079 - 39424240 = 3114015839  
}
I have gone to 11 (the first node in the rest of the network) because now we can normalize the eigenvector so that it that $v_{11}=1$.

Using this normalization we have 
\eqa{
v_1 &=& 3.2112875e-10 \\
v_2 &=& 2.89015871e-9 \\
v_3 &=& 2.56902996e-8 \\ 
v_4 &=& 2.28322538e-7 \\
v_5 &=& 0.00000202921 \\
v_6 &=& 0.00001803459 \\ 
v_7 &=& 0.0001602821 \\ 
v_8 &=& 0.00142450431 \\ 
v_9 &=& 0.01266025673 \\ 
v_{10} &=& 0.1125178063 \\ 
v_{11} &=& 1.0
}
So the eigenvector stays very well confined to the rest of the network. 

Fun fact: One eigenvalue of the network is very close to the golden ratio. Networks that contain long chains typically have such an eigenvalue.
