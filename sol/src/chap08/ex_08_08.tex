\exercise{Great expectations}{3}
Some coral reefs are close enough to be in biological contact. Typically we don't know much about the networks of species dispersal that form between them. Let's assume that in a certain area, we know that 30\% of the reefs are completely isolated. In the absence of other information, would you expect that a giant component exists?

\solution 
Let's assume the degree distribution is an ER graph, which is very reasonable for sparsely connected networks. We know the degree distribution 
\eq{
p_k = \frac{z^k\exp{-z}}{k!}, 
}
which means for the isolated nodes 
\eq{
p_0 = \frac{z^k\exp{-z}}{k!} = \exp{-z} = 0.3
}
which we can solve for
\eq{
z = -{\rm log}(p_0)= -{\rm log}(0.3) \approx 1.2,
}
in an ER graph this would imply $q\approx 1.2$ so we should expect that there is a giant component, at least until additional information arrives. 

In general, if we know nothing about the network except the proportion of isolated nodes, we would expect that there is a giant component if the proportion of isolated nodes is 
\eq{
p_0 < \frac{1}{\rm e} \approx 36\%.
}
